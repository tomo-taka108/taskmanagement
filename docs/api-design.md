## 3. HTTPステータスコード規則

| ステータス | 用途 |
|-----------|------|
| 200 OK | 取得・更新成功 |
| 201 Created | 作成成功 |
| 204 No Content | 削除成功 |
| 400 Bad Request | リクエストパラメータ不正 |
| 404 Not Found | 指定リソースが存在しない |
| 500 Internal Server Error | サーバー内部エラー |
